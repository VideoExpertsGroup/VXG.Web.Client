Camera preview with click-to-player event

Implemented in "player" module. The camera token necessary for the module to work is set using the "camtoken" attribute. If the attribute is absent in the current element, the module will look for the attribute in the parent elements
<div access_token="eyJhY..."><campreview onclick_toscreen="player"></campreview></div>
<div access_token="217757"><campreview onclick_toscreen="player"></campreview></div>
<div access_token="+1"><campreview onclick_toscreen="player"></campreview><camfield field="name"></camfield>/div>
